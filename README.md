#  ecpic - unknown

**꾸준한 도전으로 나를 바꾸자**<br/>
**challenge little by little every day to change yourself**<br/>
- To-do 리스트와 캐릭터 육성 기능이 합쳐진 게임형 자기 관리 어플리케이션입니다.<br/>
- 하루의 목표는 운동, 공부 그 무엇이든 상관없습니다.<br/>
- 자격증, 수상과 같은 활동 진행시에도 경험치 증가로 캐릭터를 육성할 수 있습니다.<br/>
